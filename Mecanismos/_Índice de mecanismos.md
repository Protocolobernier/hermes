# ⚙️ Mecanismos

Cada mecanismo se desglosa en **eslabones**, y cada eslabón se gradúa:

- `🟢 Demostrado` — evidencia directa y replicada
- `🟡 Bien fundado` — apoyo sólido pero indirecto o parcial
- `🔴 Extrapolación` — salto lógico aún no demostrado (candidato a hipótesis)
